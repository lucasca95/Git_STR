This directory contains a set of Cheddar project examples :
  • The aadl directory contains a set of AADL examples. This directory shows the AADL capabilities of Cheddar.
  • The tests directory contains a set of Cheddar projects that we use to run for each new release. This directory contains the regression tests of Cheddar.
  • The xml directory contains a set of Cheddar's XML examples. This directory shows different services provided by Cheddar.
